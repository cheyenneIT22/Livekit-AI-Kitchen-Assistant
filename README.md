# Kitchen AI Voice Assistant 

An AI-powered kitchen assistant built using:
- LiveKit Agent demo
- OpenAI API
- Async Python

## Features
- Metric units (AU friendly 🇦🇺)
- Async kitchen timers
- Conversation memory

## Setup

1. Create virtual environment
2. Install dependencies
3. Add .env file with:

LIVEKIT_URL=
LIVEKIT_API_KEY=
LIVEKIT_API_SECRET=
OPENAI_API_KEY=

4. Run:

python agent.py
